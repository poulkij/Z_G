Quick Start

三步开始使用

01
安装 Skill
在你的 AI Agent 项目中安装 zettaranc skill，支持 Claude Code 和 Cursor。
npm install -g zettaranc-skill
02
配置 Agent
将 skill 注册到你的 agent 配置中，一行命令即可完成。
zettaranc init --agent claude
03
开始对话
直接用自然语言与 agent 对话，zettaranc 会自动识别意图并调用相应功能。
claude "分析一下比亚迪最近的走势"
