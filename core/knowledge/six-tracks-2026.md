## 概述

2026 年 A 股核心逻辑：**增量资金有限 + 政策驱动明确 + 产业周期错位** = 只有部分赛道能走出独立行情。

**选股优先级**：赛道 > 个股 > 择时

## 六大赛道框架

> **注意**：以下内容基于 `market-macro.md` 和 `portfolio-management.md` 中已有的产业资本视角、新曼城 4231 体系等片段整合。具体标的和权重需根据最新语料更新。

### 赛道一：创新药 / 生物医药

**核心逻辑**：
- BD（Business Development）出海进入兑现期
- 三生制药 12.5 亿美元首付款打开创新药估值天花板
- BD 四节点炒作：蹭概念 → 获得首付款 → 里程碑付款 → 销售分成

**重点关注**：
- 五大：痛风 / 自免 / 肺癌 / 麻醉 / 胰腺癌
- 三小：阿尔茨海默 / 抑郁症 / 小细胞胰腺癌

**风险**：临床失败、医保谈判降价、出海审批不确定性

---

### 赛道二：AI 算力 / 半导体设备

**核心逻辑**：
- 国产替代进入深水区（光刻机、EDA、先进封装）
- AI 算力需求持续高增长，但需区分"真需求"与"伪概念"
- 产业资本视角：设备商先于设计公司受益

**重点关注**：
- 半导体设备（刻蚀、薄膜、检测）
- 先进封装（Chiplet、HBM）
- 国产 EDA 与 IP

**风险**：技术迭代快、国际制裁升级、估值泡沫

---

### 赛道三：新能源（储能 + 电网改造）

**核心逻辑**：
- 光伏/风电装机量持续增长，消纳瓶颈倒逼电网改造
- 储能从技术验证进入商业化放量期
- 产业资本视角：电网侧储能 > 户用储能 > 便携式储能

**重点关注**：
- 电网智能化改造
- 大储（大型储能）系统集成商
- 钠离子电池（成本替代逻辑）

**风险**：产能过剩、价格战、政策补贴退坡

---

### 赛道四：高端制造 / 工业机器人

**核心逻辑**：
- 人口红利消退 → 机器换人加速
- 人形机器人从 0 到 1，供应链率先受益
- 产业资本视角：减速器、丝杠、传感器等核心零部件

**重点关注**：
- 精密减速器（谐波、RV）
- 滚珠丝杠 / 行星滚柱丝杠
- 力矩传感器 / 视觉传感器

**风险**：技术路线不确定、量产进度慢于预期

---

### 赛道五：消费升级 / 品牌出海

**核心逻辑**：
- 国内消费分层，高端化与性价比并行
- 中国品牌从"中国制造"走向"中国品牌"
- 产业资本视角：有定价权的品牌 > 有渠道的品牌 > 纯制造

**重点关注**：
- 高端白酒 / 保健品（内需稳定）
- 小家电 / 潮玩 / 美妆（出海逻辑）
- 运动服饰 / 户外用品（赛道景气）

**风险**：消费复苏不及预期、汇率波动影响出海利润

---

### 赛道六：周期资源（铜 / 铝 / 稀土）

**核心逻辑**：
- 全球制造业复苏 + 新能源需求拉动有色金属
- 供给端受限（矿山品位下降、环保约束）
- 产业资本视角：铜 > 铝 > 稀土，按供需缺口排序

**重点关注**：
- 铜矿（新能源用铜量倍增）
- 电解铝（产能天花板已现）
- 稀土磁材（人形机器人 + 风电增量）

**风险**：全球经济衰退、美元走强压制大宗商品价格

## 动态调整原则

1. **季度复盘**：每季度评估六大赛道的产业数据是否支持原有逻辑
2. **去弱留强**：某个赛道连续两个季度没有催化剂，减仓换到更强赛道
3. **不押注单一赛道**：最多 30% 仓位在单一赛道，避免黑天鹅
4. **政策敏感度**：关注两会、政治局会议、产业政策文件，及时调整
