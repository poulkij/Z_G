## 一、2026 年行情特征总结

### 1.1 四年周期定位

| 年份 | 阶段 | 策略特征 |
|------|------|---------|
| 2024 | 捡钱年 | 大胆入场，随便买都赚 |
| 2025 | 估值修复年 | 核心资产持有，少妇战法 |
| **2026** | **投机交易年** | **屁胡哲学，做波段** |
| 2027 | 业绩抢钱年 | 基本面驱动 |

### 1.2 震荡市三大特征

| 特征 | 表现 | 应对 |
|------|------|------|
| **行情变短** | 上涨持续时间从牛市 2-4 周压缩到 3-5 天 | 买入后 3 天不涨就走 |
| **幅度变小** | 涨幅从 20-30% 压缩到 5-15% | 见红就卖，不贪大涨 |
| **波动频繁** | 日内振幅加大，一天涨一天跌 | 不追高，等回调买 |

### 1.3 市场三阶段分布

| 阶段 | 占比 | 特征 | 策略 |
|------|------|------|------|
| 垃圾时间 | 66% | 阴跌/横盘 | 少折腾，空仓为主 |
| 舒适区 | 24% | 有赚钱效应 | 敢布局，快进快出 |
| 高波动期 | 10% | 涨猛跌更猛 | 管住手，宁可错过 |

## 二、屁胡哲学完整版

### 2.1 核心定义

**屁胡 = 麻将术语，不贪大牌，见胡就赢，积小胜为大胜**

- 牛市策略：捂股不动，等主升浪
- 震荡市策略：**见红就卖，落袋为安**
- 核心理念：不追求单次暴利，追求稳定复利

### 2.2 见红就卖的具体标准

| 信号类型 | 具体标准 | 操作 |
|---------|---------|------|
| **必卖信号** | 买入后浮盈变成浮亏（盈转亏） | 立刻清仓 |
| **必卖信号** | 买入后 3 天不涨 | 开盘清仓 |
| **必卖信号** | 顶部放巨量阴线（S1） | 清仓 |
| **减仓信号** | 中大阳线（4%+）次日冲高 | 减半仓 |
| **减仓信号** | 第四块红砖走完 | 减半仓 |
| **减仓信号** | 次日阴量 > 阳量 3/4 | 减半仓 |

### 2.3 屁胡操作纪律

1. **买入后当天不涨**：次日 9:33/9:37 找高点走
2. **买入后 3 天不涨**：最多拿 4 天，第 4 天必须走
3. **盈利达到 5-8%**：考虑减半仓，剩下拿着博更大
4. **盈利达到 10%+**：可以适当格局，但破白线必须走
5. **盈利回到 5% 以内**：利润垫消失，改为优先跑

### 2.4 屁胡 vs 牛市持有对比

| 场景 | 牛市（2024-2025） | 震荡市（2026） |
|------|------------------|---------------|
| 买入后不涨 | 等，可以拿 1-2 周 | 3 天不走就割 |
| 盈利后回调 | 利润垫厚可以扛 | 5% 以内利润垫就撤 |
| 卖出时机 | 等 S1/S2 信号 | 见红就卖 |
| 目标收益 | 20-50% | 5-15% |
| 持股心态 | 格局持有 | 斤斤计较 |

## 三、空仓哲学应用场景

### 3.1 空仓定义

**止损的最高境界是空仓。空仓不是不做，是等最好的机会。**

### 3.2 必须空仓的场景

| 场景 | 判断标准 | 操作 |
|------|---------|------|
| 活跃市值翻绿 | 活跃市值 < 0 或 -2.3% 以下 | 空仓/轻仓 |
| 白线在黄线下 | 死叉状态 | 不开新仓 |
| 绿砖阶段 | 砖形图显示下跌循环 | 绝不抄底 |
| 大盘破位 | 放量跌破关键支撑 | 空仓观望 |
| 行情不明 | 无主线/无持续热点 | 休息等待 |

### 3.3 空仓期间做什么

1. **看而不动**：持续跟踪自选股，等待买点出现
2. **研究基本面**：深挖主线方向和标的
3. **等待完美图形**：等 B1、超级 B1 等标准图形
4. **保持手感**：小仓位试单，感受市场节奏

### 3.4 空仓铁律

- 「百美图」— 不急于出手，等完美标的
- 「敌不动我不动，敌进我退，敌退我追」
- 把自己定位为市场空头

## 四、震荡市选股→买入→持有→卖出完整 SOP

### 4.1 第一步：选股（选池子）

| 条件 | 具体标准 |
|------|---------|
| 趋势要求 | 黄线之上、白线之上、有砖形图节奏 |
| 图形要求 | N 型结构、异动+地量、B1 买点形态 |
| 基本面 | 十五五重点产业、核心资产 |
| 资金面 | 近期有异动放量、主力进场信号 |

**选股优先级**：
1. 纯血主线票（单一主题核心标的）
2. 回调到黄线附近的 B1
3. 超级 B1（放量下跌后缩量企稳+J 负值）

### 4.2 第二步：买入

| 买入类型 | 具体操作 |
|---------|---------|
| **B1 买入** | J ≤ -10、缩量回调到黄线/白线、分批买（5k+5k） |
| **超级 B1** | 反转十字星出现次日、止损设放量下跌线最低价 |
| **B2 确认** | 一阳穿多线、涨幅 ≥ 4%、J < 55、开盘买一半下杀补一半 |

**买入时间**：
- 牛市：早盘开盘买
- 震荡市：**下午 14:30 之后买**，避开早盘波动

**仓位控制（3-2-2 阵型）**：
- 30-35% 侦察兵（试仓）
- 20-25% 主力军（确认后加仓）
- 40-50% 预备队（留作机动）

### 4.3 第三步：持有（持有期管理）

| 时间 | 操作 | 判断标准 |
|------|------|---------|
| 买入当天 | 记录成本 | 看是否脱离成本线 |
| 第 1 天 | 观察 | 不涨不跌正常 |
| 第 2 天 | 重点观察 | 没涨就要警惕 |
| 第 3 天 | **决策日** | 没涨 = 必须走 |
| 第 4 天 | 最后机会 | 还不涨 = 清仓 |

**持有期预警**：
- 3 天不涨 → 准备走
- 盈转亏 → 立刻走
- 利润回到 5% 以内 → 准备走
- 破白线 → 必须走

### 4.4 第四步：卖出（止盈/止损）

| 卖出类型 | 具体标准 | 操作 |
|---------|---------|------|
| **止损** | 前低下 3-5 价位 / 横盘平台下沿 / 买入 K 线下方 | 三选一，跌破就走 |
| **放飞** | 中长阳（4% 中阳、7% 大阳）+ 第四块砖 | 减半仓 |
| **S1 信号** | 顶部放量阴线、假阴真阳也算 | 先信，优先走 |
| **滴滴** | 高位连续两根阴线下台阶 | 第二根走 |
| **破白线** | 跌破 EMA(EMA(C,10),10) | 牛股不破白线 |
| **破黄线** | 跌破 (MA14+MA28+MA57+MA114)/4 | 最后止损线 |

**震荡市卖出优先级**：
1. 盈转亏 → 立刻走（最高优先级）
2. 3 天不涨 → 走
3. 减仓信号出现 → 减半仓
4. 利润垫 5% 以内 → 准备走

## 五、B1 五补丁在震荡市的应用

### 5.1 B1 双补丁（基础）

| 补丁 | 规则 | 目的 |
|------|------|------|
| **补丁 1** | 周线 55/144/233 周线金叉 + 趋势向上 | 过滤下跌趋势票 |
| **补丁 2** | 前三根中大阳线累计换手率 < 38% | 过滤主力已出货的票 |

### 5.2 B1 震荡市补丁（新增）

| 补丁 | 规则 | 目的 |
|------|------|------|
| **补丁 3** | 买入区间控制在 5% 以内 | 锁定成本，避免被套 |
| **补丁 4** | 只做回调到黄线/白线的 B1 | 交易价值最大 |
| **补丁 5** | 震荡市仓位上限 5%（牛市 10%） | 控制风险 |

### 5.3 震荡市 B1 特别纪律

1. **买入后当天必须脱离成本线**：否则次日开盘走
2. **利润达到 5% 就减半仓**：落袋为安
3. **绝不在绿砖阶段买入**：等红砖再动
4. **优先超级 B1**：比普通 B1 胜率更高

## 七、过渡期策略调整

### 7.1 什么时候开始转换

当出现以下信号时，从牛市策略转向震荡市策略：

1. **活跃市值持续在 0 轴附近波动**
2. **个股上涨持续性变差**（之前涨 2 周现在只涨 3 天）
3. **涨幅普遍缩小**（之前 20-30% 现在只有 10%）
4. **Z 哥明确提示**：「今年是投机交易年」

### 7.2 过渡期操作

| 阶段 | 特征 | 操作 |
|------|------|------|
| **牛市末期** | 还在涨但涨幅变小 | 开始分批减仓 |
| **过渡期** | 涨跌交替，方向不明 | 只做 B1，快进快出 |
| **震荡市确认** | 明显区间震荡 | 全面切换屁胡策略 |

### 7.3 过渡期心理调整

- **从「格局」到「计较」**：之前看日线，现在看小时线
- **从「等信号」到「抢就跑」**：买入就要想着什么时候卖
- **从「分散持仓」到「集中持仓」**：震荡市不适合开超市
- **从「怕卖飞」到「敢于卖飞」**：卖飞是常态，不卖飞才是运气
