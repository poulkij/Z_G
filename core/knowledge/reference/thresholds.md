## 仓位管理核心阈值

### 仓位铁律（三铁律，2012年定下）

| 维度 | 阈值 | 说明 |
|------|------|------|
| 单票仓位 | ≤10% | 单只票崩盘不影响整体净值 |
| 总仓位 | ≤80% | 永远留20%现金 |
| 现金储备 | ≥20% | 市场跌了有钱补仓 |

### 市场状态分层仓位

| 市场状态 | 仓位范围 | 策略 |
|---------|---------|------|
| 强趋势牛市 | 80-100% | Beta>Alpha，满仓不动 |
| 震荡市 | 50-80% | 底仓+动态仓，单票≤20% |
| 空头区间 | 0-50% | 只卖不买 |

### 仓位分级（按波段类型）

| 波段类型 | 仓位 | 说明 |
|---------|------|------|
| 上升波段 | 10% | 高确定性 |
| 震荡波段 | 5% | 中等确定性 |
| 下跌波段 | 2-3% | 低确定性 |

### 建仓阵型（3-2-2）

| 仓位 | 用途 | 说明 |
|------|------|------|
| 30-35% | 侦察兵 | 试仓 |
| 20-25% | 主力军 | 确认后加仓 |
| 40-50% | 预备队 | 机动 |

### 底仓+动态仓

| 仓位 | 用途 |
|------|------|
| 40-50% | 底仓 |
| 30-40% | 动态仓 |

## 买入信号阈值

### B1量化标准

| 指标 | 阈值 | 说明 |
|------|------|------|
| J值 | ≤-10（最好负值） | KDJ参数(9,3,3) |
| 涨幅 | -2% ~ 1.8% | 十张完美图形统计 |
| 振幅 | 7%以内 | 越小越好 |
| 累计换手率 | <38% | 超过就走 |
| 自由换手率 | <62% | L2数据 |

### B2确认条件

| 条件 | 阈值 |
|------|------|
| B1后涨幅 | ≥4% |
| B1后交易日 | 3日内 |
| KDJ钩值 | <55 |
| 放量要求 | 比前一日放量 |
| 上影线 | 无上影线最佳 |

### B3确认条件

| 条件 | 阈值 |
|------|------|
| 前置条件 | B2后十字星/小阴线 |
| 开盘 | 平开一致 |
| 止损位置 | B2阳线中间或B3当天最低 |

| 量比 | 场景 | 操作 |
|------|------|------|

|------|------|------|
| S1 | 丑陋大绿帽+放量 | 100亿以下小票清仓，大票减半 |
| S2 | 挑S1前高+MACD顶背离 | 清仓/大幅减仓 |
| S3 | 反抽巨量下沿 | 最后逃生机会 |

### 滴滴战法

| 条件 | 操作 |
|------|------|
| 14:55+跌破昨低+浮盈 | 飞一半 |
| 14:55+跌破昨低+浮亏 | 全清仓 |

### BBI离场

- 收盘价连续两天跌破BBI → 清仓
- 牛市可破白线，绝不可破黄线
- 有利润垫(10%+)可扛到黄线

### 防卖飞评分（5分制）

| 评分 | 条件 | 操作 |
|------|------|------|
| 4-5分 | 收盘涨+BBI没破+非放量阴线+趋势向上+J未死叉 | 持有/减仓 |
| 3分 | 减一半，等BBI两日破位 | 中庸 |
| <3分 | 准备离场 | 离场 |

|------|------|------|

## 资金管理公式

**最大回撤与仓位关系**：
- 最多接受5%回撤
- 遇到两个跌停(20%)
- 仓位降到1/4

**止损计算**：
- 阴线最低价下方2-3%
- 止损2-3%，止盈10-20%
- 盈亏比约3:1
