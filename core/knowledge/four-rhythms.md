## 四大节奏概述

| 节奏 | 主力行为 | 散户应对 | 关键识别 |
|------|---------|---------|---------|
| **建仓节奏** | 吸筹、洗盘、拿筹码 | 跟随 B1 买入 | 低位放量、N 型抬高、筹码密集 |
| **拉升节奏** | 脱离成本、吸引跟风 | 持有，让利润飞 | 量价齐升、白线在上、呼吸健康 |
| **洗盘节奏** | 震仓、清理浮筹 | 防卖飞评分 ≥4 则持有 | 缩量回调、不创新低、关键 K 守住 |
| **出货节奏** | 派发、兑现利润 | S1/S2/S3 信号出现即走 | 放量阴线、绿肥红瘦、顶部大风车 |

---

## 建仓节奏

**时间跨度**：数周至数月

**特征**：
1. 底部区域反复震荡，成交量间歇性放大
2. N 型结构逐步抬高，低点不断抬升
3. 阴线缩量、阳线放量（红肥绿瘦）
4. 关键 K 线出现：底部放量长阳、单针探底

**最佳介入点**：
- J 负值 B1（日线或周线）
- 缩量回调到白线/黄线附近
- 放量突破平台后的缩量回踩

---

## 拉升节奏

**时间跨度**：数日至数周

**特征**：
1. 沿白线或 BBI 流畅上涨，回踩即反弹
2. 量价配合：上涨放量、回调缩量
3. 涨停或中大阳线居多，阴线短而小
4. MACD 红柱持续，DIF 在零轴上方

**持有纪律**：
- 不破白线，不止盈
- 连续两根中/大阳线 → 减半（卤煮）
- 出现 S1 信号 → 直接走

---

## 洗盘节奏

**时间跨度**：数日

**特征**：
1. 缩量回调，成交量明显萎缩
2. 回调幅度有限，不跌破关键支撑（白线/黄线/BBI）
3. 可能出现「灾区重建」形态：放量金叉后缩量回踩黄线
4. 假阴真阳、长下影线等洗盘 K 线

**应对**：
- 防卖飞 V1.4 评分 ≥4 → 坚定持有
- 评分 2-3 → 减仓观察
- 评分 ≤1 → 按纪律止损

---

## 出货节奏

**时间跨度**：数日至数周

**特征**：
1. 高位放量滞涨或放量下跌
2. 绿肥红瘦，阴量大于阳量
3. 顶部出现丑陋大绿帽、大风车、双头等形态
4. MACD 顶背离

**纪律**：
- S1 出现 → 至少卖一半（小票全清）
- S2 出现 → 无条件清仓或大幅减仓
- S3 出现 → 最后逃生窗口，错过不再

---

## 节奏切换的信号

| 从 → 到 | 信号 |
|---------|------|
| 建仓 → 拉升 | 放量突破平台 + 缩量回踩确认 |
| 拉升 → 洗盘 | 缩量回调 + 关键支撑守住 |
| 洗盘 → 拉升 | 地量后放量长阳（B2 确认） |
| 拉升 → 出货 | S1  ugly 大绿帽 / 放量阴线 |
| 出货 → 回落 | 跌破所有支撑 + 缩量无承接 |
| 回落 → 建仓 | 地量地价 + 底部筹码重新密集 |
